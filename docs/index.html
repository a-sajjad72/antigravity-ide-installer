Built for Stability & Ease of Use

Unlike generic scripts, this installer packages professional, production-grade safety mechanisms to ensure your development environment remains pristine.

Atomic Swaps

Installs updates cleanly into a temporary workspace. Swaps folders only when successful, eliminating corruption from interrupted downloads.

Auto Backups

Automatically compresses your settings, chat histories, agent sessions, and extensions into backups before updating.

Legacy Profile Migration

Transitioning from Google's APT repos? Your settings are automatically detected, permissions fixed, and migrated seamlessly.

Quick Rollback

Encountered a bug in the latest version of the IDE? Instantly toggle back to your previously active installation using a single CLI argument.

Clean Uninstallation

If you ever need to clean your system, you can completely remove the installer, links, binaries, and icons in seconds.

sudo rm -rf /opt/antigravity-ide
sudo rm -f /usr/local/bin/antigravity-ide
sudo rm -f /usr/local/bin/update-antigravity-ide
sudo rm -f /usr/share/applications/antigravity-ide.desktop
sudo rm -f /usr/share/icons/hicolor/512x512/apps/antigravity-ide.png

# Update local desktop environment caches
sudo update-desktop-database /usr/share/applications 2>/dev/null || true
sudo gtk-update-icon-cache -q /usr/share/icons/hicolor 2>/dev/null || true
